# Dataset and Private Files

This release excludes:

- FaceScape raw meshes and scans
- HRN weights and generated HRN mesh caches
- OBJ, PLY, STL, NPY, NPZ, PKL, PT, and PTH outputs
- Private server utilities and local machine configuration
- Exploratory experiment archives that are not used by the manuscript tables

Prepare the dataset separately and set the `PAPERFIT_ROOT` environment variable before rerunning experiments.
